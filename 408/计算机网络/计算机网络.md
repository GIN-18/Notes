# 计算机网络

计算机网络是一个将分散的，具有独立功能的`计算机系统`，通过`通信设备`与`线路`连接起来，由完善的`软件`实现`资源共享`和`信息传递`的系统。

计算机网络是一个`互连`的，`自治`的计算机集合。

## 计算机网络的组成

组成部分

```
硬件，软件，协议
```

工作方式

```
边缘部分：用户可以指直接使用(C/S方式，P2P方式)

核心部分：为边缘部分服务
```

功能组成

```
通信子网：实现数据通信

资源子网：实现数据共享/数据处理
```
