## let 和 const

----------

`let` 和 `const` 声明的变量有如下特点：

1. 不存在变量提升

2. 不允许重复声明

3. 具有块级作用域

4. 暂时性死区

#### 暂时性死区

只要块级作用域内存在 `let` 命令，则 `let` 命令声明的变量就 `绑定` 在这个区块内，不再受外部影响。

```javascript
var a = 1 ;

{
  console.log(a);
  let a = 2;
}
```

`let`声明的变量`绑定`在这个区块内，区块内的打印语句访问不到外部声明的变量`a`。

## 变量的解构赋值

----------

### 数组的解构赋值

---

基本用法:

```javascript
let [foo, [bar, baz]] = [1, [2, 3]];

foo // 1
bar // 2
baz // 3
```

### 对象的解构赋值

---

基本用法：

```javascript
let { foo, bar } = { foo: "aaa", bar: "bbb" };

foo // aaa
bar // bbb
```

### 字符串的解构赋值

---

基本用法：

```javascript
let [a, b, c, d, e] = "hello";

a // h
b // e
c // l
d // l
e // o
```

## 模板字符串

---

模板字符串是增强版的字符串，用 `反引号` 标识。

模板字符串可以当作普通字符串使用，也可以用来定义多行字符串，或者在字符串中嵌入变量。
